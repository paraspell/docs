## Prerequisites
- Node.js
- pnpm

## Launch locally

```bash
# Clone XCM Tools
$ git clone https://github.com/paraspell/xcm-tools.git

# Install monorepo packages
$ pnpm install

# Build monorepo projects
$ pnpm build

# Run playground via monorepo script
$ pnpm run:playground

```
